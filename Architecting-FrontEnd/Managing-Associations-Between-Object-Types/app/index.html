Foreign Language Vocabulary Training

A Plain JS Web App with Unidirectional Associations

  • People
  • Translation Problems
  • Learning Units

This is the 4th example app of the Bov Academy course Architecting-Modern-Frontend-Web-Applications. The app includes the following data management options:

  • Manage person data
  • Manage translation problems
  • Manage learning units
    •
  • test data
  • database
    •

    For playing with the app, you may first want to and then view it (e.g. by choosing "Manage learning units" and then "Retrieve and list all learning units"). For observing the internal status and error messages, turn on the JavaScript console.

    This front-end data management app takes care of three object types (people, translation problems and learning units) as well as the two unidirectional associations that assign an author and several translation problems to a learning unit. The app supports the four standard data management operations (Create/Retrieve/Update/Delete).
